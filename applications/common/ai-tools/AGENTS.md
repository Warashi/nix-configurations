- git add --force は禁止です
- git-mile の task を実施する際には、実施前と完了後に task を適切に更新してください
- git-mile に task を追加する際には、task の目的、完了条件などを明確に記載してください
- git-mile の task は細かい作業単位で追加してください
- git-mile の task は無限に階層分けできます。適切に階層分けしてください。
